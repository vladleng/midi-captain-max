# Contributing to MIDI Captain MAX

Thanks for your interest in contributing! This project is maintained by [@mcascone](https://github.com/mcascone) and welcomes help from the community.

---

## Ways to Contribute

### Bug Reports
If something isn't working, open an issue. Please include:
- Which version of the MCM pedal you're using
- Your DAW / host software and version
- What you expected to happen vs. what actually happened
- Steps to reproduce the problem
- Any relevant MIDI logs or screenshots

### Feature Requests
Open an issue and describe:
- What you're trying to accomplish
- Why existing functionality doesn't cover it
- Any hardware/firmware constraints you're aware of

### Code Contributions
Before writing code, open an issue or comment on an existing one so we can discuss the approach. This avoids duplicated effort and makes PRs easier to merge.

---

## Hardware Support

Supporting new pedal versions requires physical hardware for reverse engineering. If you own a variant that isn't yet supported and want to help, open an issue and let's talk. **Do not purchase hardware on behalf of the project without checking first.**

---

## Submitting a Pull Request

1. Fork the repo and create a branch from `main`
2. Keep changes focused — one fix or feature per PR
3. Test on hardware if at all possible
4. Write a clear PR description explaining what changed and why
5. Reference any related issues with `Closes #123` or `Related to #123`
